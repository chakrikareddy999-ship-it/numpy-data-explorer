# NumPy Data Explorer
## Project Description
This project demonstrates the fundamentals of NumPy using Python.
It covers:
- Array creation
- Indexing and slicing
- Mathematical operations
- Axis-wise operations
- Statistical operations
- Reshaping and broadcasting
- Save and load NumPy arrays
- Performance comparison between NumPy and Python lists

## Technologies Used
- Python
- NumPy
- VS Code

## How to Run
1. Install NumPy:
   pip install numpy
2. Run the file:
   python numpy_data_explorer.py

## Learning Outcome
This project shows how NumPy makes numerical operations faster and more efficient compared to standard Python lists.
