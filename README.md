# numpy-data-explorer
A Python project demonstrating NumPy array creation, operations, and performance comparison with Python lists.
## Project Overview
NumPy Data Explorer demonstrates:
- 1D Array Creation
- 2D Array Creation
- Array using arange()
- Array indexing & slicing
- Basic mathematical operations
- Performance comparison (Python list vs NumPy array)
This project is built to strengthen core NumPy fundamentals.

## Technologies Used
- Python 3.x
- NumPy

## How to Run the Project

1. Clone the repository:
   git clone https://github.com/chakrikareddy999-ship-it/numpy-data-explorer.git
2. Navigate into the folder:
   cd numpy-data-explorer
3. Install dependencies:
   pip install -r requirements.txt
4. Run the file:
   python numpy_data_explorer.py

## Features Demonstrated
- Array creation using np.array()
- Array creation using np.arange()
- Multi-dimensional arrays
- Performance comparison
- Structured console output

## Learning Purpose
This project was created to practice NumPy fundamentals and understand performance differences between Python lists and NumPy arrays.
